Sum of Two Numbers

#variables #user input
Write a program that takes two numbers as input from the user and prints their sum.
Example Input Expected Output
5, 7 12

Solution

Approach

We can solve this problem by taking two numbers as input from the user, adding them, and then printing the sum.

Algorithm:

  • Step 1: Start the program.
  • Step 2: Declare variables num1, num2 and sum.
  • Step 3: Take the value of num1 as input from the user.
  • Step 4: Take the value of num2 as input from the user.
  • Step 5: Add num1 and num2 and store the result in sum.
  • Step 6: Print "The sum of num1 and num2 is sum."
  • Step 7: End the program.

Code:

#include 

int main() {
   int num1, num2, sum;

   printf("Input the first number: ");
   scanf("%d", &num1);

   printf("Input the second number: ");
   scanf("%d", &num2);

   sum = num1 + num2;

   printf("The sum of %d and %d is %d.", num1, num2, sum);

   return 0;
}

explanation:

  • In the code, we first declare three integer variables num1, num2, and sum.
  • We then use the printf() function to ask the user to input the first number and store it in num1 using the scanf() function.
  • Similarly, we ask the user to input the second number and store it in num2 using the scanf() function.
  • We then add num1 and num2 and store the result in sum.
  • Finally, we use the printf() function to print the sum along with the input values num1 and num2 in the required format.
