1)Hello, world!*
 simple hello world C program
!code
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
Here's a brief explanation of what the code does:

#include <stdio.h> is a preprocessor directive that includes the standard input/output library, which provides the printf() function for printing to the console.
int main() is the main function of the program, which is required in all C programs. 
It returns an integer value to the operating system to indicate whether the program executed successfully.
printf("Hello, world!\n"); is a function call that prints the string "Hello, world!" followed by a newline character (\n) to the console.
return 0; is a statement that exits the program and returns the integer value 0 to the operating system, which indicates that the program executed successfully.




2)char data type*
  a character data type c program
!code

    char c = 'A';
    printf("The character is %c\n", c);
    return 0;
}
In this program, we declare a char variable named c and initialize it to the character 'A'. 
We then use the printf function to print out the value of c using the %c format specifier. This will output the character 'A' to the console. 
Finally, we return 0 to indicate that the program has completed successfully.

Note that char is a data type in C that represents a single character. 
Characters are enclosed in single quotes (e.g. 'A', 'B', 'C', etc.) and are typically used to represent letters, digits, or symbols.

4)Simple Math Operations.*
  Two number being added, subtracted, multiplied and divided 
!code
#include <stdio.h>

int main()
{
    int number1, number2;
    number1 = 12;
    number2 = 4;

    printf("number1 + number2 = %d\n", number1 + number2);
    printf("number1 - number2 = %d\n", number1 - number2);
    printf("number1 * number2 = %d\n", number1 * number2);
    printf("number1 / number2 = %d\n", number1 / number2);

    return 0;
}
This code declares two integer variables (number1 and number2), initializes them to the values 12 and 4, respectively, and then performs four basic arithmetic operations on them using the +, -, *, and / operators.
The printf function is used to print the results of each operation to the console. 
The format specifier %d is used to indicate that an integer value should be printed in place of the specifier, and the escape sequence \n is used to insert a newline character after each line.
The subtraction operation (number1 + number2) add the value of number1 with number2,
The subtraction operation (number1 - number2) subtracts the value of number2 from number1, and 
the multiplication operation (number1 * number2) multiplies the values of number1 and number2 together.
The division operation (number1 / number2) divides the values of number1 and number2.

7)Celsius to Fahrenheit Conversion.*
  c program that will take temparature in celsius as input and provide output in fahrenheit units
!code
#include <stdio.h>

int main() {
    float celsius, fahrenheit;

    printf("Enter temperature in Celsius: ");
    scanf("%f", &celsius);

    fahrenheit = (celsius * 9 / 5) + 32;

    printf("%.2f Celsius = %.2f Fahrenheit\n", celsius, fahrenheit);

    return 0;
}
In this program, we first declare two variables celsius and fahrenheit of type float. 
Then, we prompt the user to enter the temperature in Celsius using the printf() function and read the input value using the scanf() function.

We then convert the temperature from Celsius to Fahrenheit using the formula (celsius * 9 / 5) + 32 and store the result in the fahrenheit variable.

Finally, we output the converted temperature using the printf() function, with formatting specifiers to show the Celsius and Fahrenheit values rounded to 2 decimal places.

Note: Make sure to include the stdio.h header file to use the printf() and scanf() functions.

27)C Program with continue statement

!code

int main() {
    int i;

    // Print numbers from 1 to 10, skipping even numbers
    for (i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            continue; // Skip even numbers
        }
        printf("%d\n", i);
    }

    return 0;
}

28)Multiplication Table Program.
a multiplication table C Program using for loop
!code
#include <stdio.h>

int main() {
    int i, j, rows, columns;
    
    printf("Enter number of rows and columns: ");
    scanf("%d %d", &rows, &columns);
    
    for (i = 1; i <= rows; i++) {
        for (j = 1; j <= columns; j++) {
            printf("%d\t", i * j);
        }
        printf("\n");
    }
    
    return 0;
}
This program prompts the user to enter the number of rows and columns they want in their multiplication table. It then uses two nested for loops to print out the multiplication table. The outer for loop runs rows times and the inner for loop runs columns times. For each row and column, it calculates the product of the row and column numbers and prints it out with a tab character \t in between. After printing out all the numbers in a row, it moves to the next row by printing a newline character \n.
